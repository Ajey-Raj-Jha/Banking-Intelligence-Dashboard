# 🏦 Banking Analytics Dashboard | End-to-End Data Analysis using Power BI

Developed an end-to-end banking analytics solution to transform raw customer financial data into meaningful business insights through interactive visual dashboards. The project covered the complete analytics pipeline including database handling, data preprocessing, exploratory analysis, and dashboard development.

## 📌 Project Pipeline

Raw Data → MySQL Database → Data Transformation → Exploratory Analysis → Interactive Power BI Dashboard

## 📊 Dataset Overview

* Dataset containing **24 attributes**
* Data managed and queried using **MySQL**
* Processed and visualized using **Power BI**

## 🔧 Data Preparation & Transformation

* Created **income segmentation categories** (Low, Mid, High) to improve customer analysis.
* Cleaned and standardized categorical fields such as **gender, nationality, and branch information**.
* Built **custom conditional columns in Power BI** for dynamic categorization.
* Converted coded values into readable business labels for better reporting and usability.
* Performed preprocessing to improve data consistency and dashboard readability.

## 📈 Exploratory Data Analysis (EDA)

Conducted detailed analysis across customer and financial dimensions:

### Customer Attributes

* Gender distribution
* Nationality trends

### Financial Metrics

* Credit Card Balance
* Loan Amount
* Deposits
* Checking Account Balance
* Savings Account Balance
* Estimated Income
* Superannuation Savings

## 💡 Major Findings

* Observed strong relationships among **deposits, savings, checking accounts, and foreign currency holdings**.
* Identified patterns showing that customers maintaining higher balances in one financial product often demonstrated stronger engagement across multiple banking services.
* Generated insights to support customer profiling and financial behavior analysis.

## 📊 Dashboard Components

* **Home Dashboard** – High-level banking KPIs and overview
* **Loan Insights** – Loan-related trends and customer behavior
* **Deposit Insights** – Deposit analysis and account performance
* **Executive Summary** – Consolidated business insights
